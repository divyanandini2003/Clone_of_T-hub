Tooltip

The Tooltip plugin is small pop-up box that appears when the user moves the mouse pointer over an element: Hover over me Hover over me. Tip: Plugins can be included individually (using Bootstrap's individual "tooltip.js" file), or all at once (using "bootstrap.js" or "bootstrap.min.js"). To create a tooltip, add the data-toggle="tooltip" attribute to an element. Use the title attribute to specify the text that should be displayed inside the tooltip:
